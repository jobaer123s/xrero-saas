SaaS platforms provide applications that run over the internet, without the need to install and run software on your computer. However, One of the most important concerns of SaaS is the protection of the stored customer data.

When it comes to enhancing the protection of your SaaS data, you must include a SaaS backup solution that will cover all the parts of data protection that your SaaS module misses. Moreover, SaaS providers themselves suggest using third-party SaaS backup software to ensure critical data is protected properly through backup, and is always available via restore.

Now, you can easily manage backup processes for your Odoo Saas Kit. Odoo Saas Kit Backup Module allows clients to create a backup process for their instance by configuring the backup cycle. The client can configure a backup cycle to create backup as daily, weekly, monthly, yearly.

Benefits of Odoo SaaS Backup Solution
Odoo SaaS backup software refers to technology that stores and protects data that SaaS products create.
  • Smart businesses use SaaS backup and restore to make sure that important data is safe.
  • Regular data backups help businesses know the problems with their data and find opportunities to fix them.
  • You do not need to suffer the disruption and downtime of restoring whole data sets to earlier states when you can identify and restore only the problematic data.

    • Detailed Features List

    • Clients can create a backup process for their instance by configuring the backup cycle.
    • Clients can configure a backup cycle to create backups daily, weekly, monthly, yearly.
    • As backup starts, the list of backups will be visible on the contract portal page
    • Clients can download the backuped file from the contract portal page.
    • Customer View
    • Admin View
    Client can find the Backup Panel by client on Backup Tab, then can create Backup Process by Click on Create Backup Process button.
    Enter Backup Process Frequency details, also select the starting date and time for backup starting.
    Client can update or cancel the backup process from the given buttons.
    Backup will list on the same portal page, client can download the backuped zip by clicking on Download button.
    After Creating by the client, backup process will be visible at client record.
    After checking the details entered by the client, admin have to confirm the process to start the backup process.
    Admin can also download the backuped database from the listed backups.
